For this project, you can use the workspace below. 
Start with cloning the Github repo into the workspace.
```
git clone https://github.com/udacity/build-ml-pipeline-for-short-term-rental-prices.git
```


**NOTE:**  
When you start working with W&S projects, you will have to export the W&S API key in each Terminal session as:
```
export WANDB_API_KEY=<your_api_key>
```
Review the  https://docs.wandb.ai/models/quickstart for an example on how to use the API key. You can go to https://wandb.ai/settings to create a personal API key owned by your user ID.
